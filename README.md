## Getting started

Run the following commands to get stated. The application uses `supabase` and the url and secret key should be specified in an `.env` file with the keys '`VITE_SUPABASE_URL` and `VITE_SUPABASE_KEY`'
```bash
git clone repo
cd codecount
npm install
npm run dev
```
